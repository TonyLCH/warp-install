# 一鍵安裝WARP腳本(為IPv6伺服器添加IPv4)

```bash
apt install wget
```
#純IPV6
```bash
wget -N --no-check-certificate "https://www.hkcls-network.gq/warp.sh" && chmod +x warp.sh && ./warp.sh
```
#純IPV4
```bash
wget -N --no-check-certificate "https://www.hkcls-network.gq/ipv4.sh" && chmod +x warp.sh && ./warp.sh
```
